## Hook: PreToolUse — unsubstituted `{{KEY}}` validation

**Source**: shanraisshan/claude-code-best-practice (fetch 2026-05-11)
**Priority**: Medium
**Reason**: The plugin uses plain text substitution on `{{KEY}}`. A forgotten placeholder (typo, unknown key added to a template but not to SKILL.md) would silently end up written into a user file. This hook would catch the problem **before** the write.

### Snippet to add in `.claude/hooks/` or `.claude/settings.json`

```javascript
// .claude/hooks/PreToolUse.js
// Detects unsubstituted {{KEY}} placeholders before Write

if (tool.type === 'filesystem.write' || tool.type === 'Write') {
  const content = tool.content || '';
  const unknownKeys = content.match(/\{\{[A-Z_]+\}\}/g) || [];
  if (unknownKeys.length > 0 && !context.allowUnknownKeys) {
    return {
      allowed: false,
      reason: `Unsubstituted placeholders: ${[...new Set(unknownKeys)].join(', ')}`
    };
  }
}
```
